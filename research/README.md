# 研究资料

本目录收集 Agent Harness 相关的理论研究、论文、实验和报告。

研究资料不只保存链接，还应沉淀为研究卡片：问题、方法、结论、局限，以及与 Harness 架构的关系。

## 目录

| 子目录 | 内容 |
|---|---|
| `papers/` | 论文、技术报告、研究文章 |
| `experiments/` | 可复现实验、对比实验、A/B 测试设计 |
| `theory/` | 控制论、软件工程、人机协作、分布式系统等理论基础 |

## 推荐研究卡片结构

- 基本信息
- 核心问题
- 主要结论
- 与 Agent Harness 的关系
- 可复用观点
- 局限性
- 原始链接
