# 标准与协议

本目录整理 Agent Harness 相关的标准、协议、接口规范和可迁移能力格式。

这里关注相对稳定的协议层和抽象层，例如 MCP、AGENTS.md、Skills、Tool Calling、多 Agent 通信与协作协议等。

## 目录

| 子目录 | 内容 |
|---|---|
| `mcp/` | Model Context Protocol 相关资料 |
| `agents-md/` | AGENTS.md 等 Agent 指令文件规范 |
| `skills/` | Skill 格式、触发、封装和迁移 |
| `tool-calling/` | Tool schema、函数调用、错误处理 |
| `multi-agent/` | Handoff、agent-as-tool、supervisor-worker 等协作模式 |
