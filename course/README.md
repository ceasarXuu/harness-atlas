# 主线课程

主线课程是 Harness Atlas 的核心内容区，负责系统讲解 Agent Harness 的基础概念、架构组件、协议标准、工程实践与未来演化。

本目录内容追求长期稳定，不作为新闻流使用。产品更新、行业动态和研究论文应先进入 `timeline/`、`products/` 或 `research/`，经过抽象后再沉淀到课程中。
