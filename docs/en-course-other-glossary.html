Bilingual Glossary

The glossary comes from the official course resources and keeps core concepts such as Agent Harness, context boundary, action boundary, runtime control, tool gateway, skill, workflow, trace, evaluation, and least agency consistent.

Bilingual Glossary

Agent Harness

  • The engineering control system around an agent.

Context Boundary

  • The information the agent is allowed to see at a step.

Action Boundary

  • The scope of actions the agent may perform on the external world.

Runtime Control

  • Mechanisms that manage multi-step execution, retry, rollback, stopping, and human intervention.

Tool Gateway

  • The unified control entry point for all tool calls.

Skill

  • A package of context, tools, steps, constraints, examples, and evaluation for a task class.

Workflow

  • A deterministic scaffold of explicit steps, branches, and validation.

Trace

  • The complete observable record of an agent run.

Evaluation

  • A feedback system that turns agent behavior into quality evidence.

Least Agency

  • Give the agent only the minimum autonomy required for the current task.
