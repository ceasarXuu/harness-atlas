统一 Agent Harness 相关术语解释。

术语表用于统一概念边界，避免 Agent、Harness、Tool、Skill、Workflow、MCP、Subagent 等概念混用。

TERMS

核心术语

AGENT

Agent

基于模型、上下文、工具和反馈执行任务的行为主体。

HARNESS

Harness

围绕 Agent 构建的运行时、控制、编排和治理层。

TOOL

Tool

暴露给 Agent 的外部能力，例如 Shell、文件、浏览器或 API。

SKILL

Skill

可复用任务能力封装，包含触发条件、步骤、资源和示例。

MCP

MCP

连接 AI 应用、工具、资源和外部系统的协议层。

SUB

Subagent

在主 Agent 或 Harness 编排下执行局部任务的 Agent。
