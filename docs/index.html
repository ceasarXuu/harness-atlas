Harness
Atlas

面向 Agent Harness 的公开知识图谱与系统课程。用软件工程视角解释 AI Agent 背后的运行环境、架构模式与工程实践。

项目关注的不是单一工具，而是 AI Agent 产品背后的通用工程范式：如何把上下文、工具、状态、记忆、规划、评估、人类在环和治理机制组合成可运行、可控制、可评估的 Agent 系统。

详细页面当前以中文维护，英文入口保持同样结构和导航边界。
