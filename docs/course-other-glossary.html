中英术语表

术语表来自正式课程资源，用于统一 Agent Harness、上下文边界、动作边界、运行时控制、工具网关、技能、工作流、跟踪、评测和最小自主权等核心概念。

中英术语表

智能体控制外壳

  • 围绕 Agent 建立的工程控制系统。

上下文边界

  • Agent 在某一步被允许看到的信息范围。

动作边界

  • Agent 被允许对外部世界执行的动作范围。

运行时控制

  • 管理多步执行、重试、回滚、停止和人工介入的机制。

工具网关

  • 所有工具调用的统一控制入口。

技能

  • 围绕一类任务封装的上下文、工具、步骤、约束、示例和评测。

工作流

  • 显式步骤、分支和校验构成的确定性支架。

跟踪

  • 一次 Agent run 的完整可观测记录。

评测

  • 把 Agent 行为转化为质量证据的反馈系统。

最小自主权

  • 只给 Agent 完成当前任务所需的最小自主空间。
