# Bilingual Glossary / 中英术语表

## Agent Harness / 智能体控制外壳
- 中文：围绕 Agent 建立的工程控制系统。
- English: The engineering control system around an agent.

## 上下文边界
- 中文：Agent 在某一步被允许看到的信息范围。
- English: The information the agent is allowed to see at a step.

## 动作边界
- 中文：Agent 被允许对外部世界执行的动作范围。
- English: The scope of actions the agent may perform on the external world.

## 运行时控制
- 中文：管理多步执行、重试、回滚、停止和人工介入的机制。
- English: Mechanisms that manage multi-step execution, retry, rollback, stopping, and human intervention.

## 工具网关
- 中文：所有工具调用的统一控制入口。
- English: The unified control entry point for all tool calls.

## 技能
- 中文：围绕一类任务封装的上下文、工具、步骤、约束、示例和评测。
- English: A package of context, tools, steps, constraints, examples, and evaluation for a task class.

## 工作流
- 中文：显式步骤、分支和校验构成的确定性支架。
- English: A deterministic scaffold of explicit steps, branches, and validation.

## 跟踪
- 中文：一次 Agent run 的完整可观测记录。
- English: The complete observable record of an agent run.

## 评测
- 中文：把 Agent 行为转化为质量证据的反馈系统。
- English: A feedback system that turns agent behavior into quality evidence.

## 最小自主权
- 中文：只给 Agent 完成当前任务所需的最小自主空间。
- English: Give the agent only the minimum autonomy required for the current task.
